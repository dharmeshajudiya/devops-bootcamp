# DevOps Bootcamp Video Player

A modern, interactive video player for DevOps Bootcamp course with sidebar navigation and enhanced UI.

## Features

- **Sidebar Navigation**: Collapsible sections for easy course navigation
- **Video Player**: Full-screen video player that fits to screen
- **Toggle Sidebar**: Hide/show sidebar with a round toggle button
- **Play Icons**: Visual indicators (▶ for play, ⏸ for playing in green)
- **Previous/Next Navigation**: Easy navigation between lessons
- **Auto-play**: Automatically plays next lesson when current one ends
- **Active State**: Highlights currently playing lesson

## Local Setup

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- No server required - runs directly in browser

### Installation

1. Clone the repository:
```bash
git clone https://github.com/dharmeshajudiya/devops-bootcamp.git
cd devops-bootcamp
```

2. Open the HTML file:
   - Simply double-click `devopsbootcamp.html` in your file explorer, or
   - Right-click `devopsbootcamp.html` → Open with → Your preferred browser

### Usage

1. **Browse Lessons**: Scroll through the left sidebar to see all available modules and lessons
2. **Play Video**: Click on any lesson to start playing
3. **Toggle Sidebar**: Click the round button (< or >) at the top-left to hide/show the sidebar
4. **Navigate**: Use Previous/Next buttons below the video player
5. **Collapse Sections**: Click on any module header to collapse/expand that section
